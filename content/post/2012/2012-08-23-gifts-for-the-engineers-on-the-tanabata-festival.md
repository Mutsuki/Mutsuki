Tips:   
ASCII 字母表里的A用二进制表示 01000001  
A = 01000001  
B = A + 1  
B = 01000001 + 1 = 01000001 + 00000001 = 01000010  
C = B + 1  
C = 01000010 + 1 = 01000010 + 00000001 = 01000011  
Z = A + 25 = 01000001 + 00011001 = 01011010
